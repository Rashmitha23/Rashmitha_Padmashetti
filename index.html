Air Quality Prediction Using Flask

Developed a machine learning-based web application to assess and predict air quality, by leveraging air pollutant data (e.g., PM2.5, ammonia, NOx, ozone, CO, etc.), using regression models to predict the Air Quality Index (AQl). Through Random Forest and Decision Tree regression, the Random Forest algorithm provided superior accuracy for AQI predictions, achieving 98.98% accuracy. To make this accessible, I created a user-friendly web application using Flask, where users can input pollutant data to receive realtime air quality insights, categorized from Good to Very Poor.

Aging Sign Detection.

Collaborated with a team to develop a machine-learning model to detect two common signs of ageing: wrinkles and dark spots. Using OpenCV, Numpy, and Matplotlib, we processed facial images by first converting them to grayscale. Edge detection identified wrinkle patterns based on threshold counts for wrinkles. For dark spots, thresholding and contour analysis pinpointed spot locations. If wrinkles or dark spots exceeded set thresholds, they were flagged as present.
